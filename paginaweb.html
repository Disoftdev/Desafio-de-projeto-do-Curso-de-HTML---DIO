Breve História da Web

(Voltar!)

A história da web é como uma jornada de evolução tecnológica. Tudo começou nos anos 1960, com a criação da ARPANET, uma rede que conectava universidades e centros de pesquisa nos EUA. Mas foi só nos anos 1990 que a web, como a conhecemos realmente nasceu. Tim Berners-Lee um cientista britânico que criou a World Wide Web em 1989 enquanto trabalhava no CERN, um centro de pesquisas na Suíça.

Ele desenvolveu três tecnologias fundamentais: o HTML (a linguagem que estrutura as páginas da web), o URL (o endereço de um site) e o HTTP (o protocolo que permite a troca de informações entre computadores). Em 1991, ele lançou o primeiro site. Nos anos seguintes, a web cresceu rapidamente. Navegadores como o Mosaic e, depois, o Netscape, facilitaram o acesso à internet, tornando-a popular.

Nos anos 2000, a web 2.0 trouxe uma internet mais interativa, com redes sociais, blogs e conteúdo gerado por usuários. Hoje, a web é muito mais do que apenas páginas estáticas. Temos a web 3.0, que promete ser mais inteligente, com o uso de inteligência artificial, blockchain, e muito mais. É um mundo em constante evolução, e agora significa que você está bem no meio dessa revolução!

Estrutura básica do HTML

(Voltar!)

HTML é a linguagem usada para criar páginas web, estruturando o conteúdo através de tags. A estrutura básica inclui "html", que envolve todo o código; "head", onde ficam metadados e o título da página; e "body", onde está o conteúdo visível, como textos, imagens e links. Cada parte do conteúdo é organizada com tags específicas, como "h1" para cabeçalhos e "p" para parágrafos. Essa estrutura forma o esqueleto de qualquer página na internet.

Falando sobre Tags.

(Voltar!)

Tags são os elementos básicos do HTML que definem e organizam o conteúdo de uma página web. Elas são escritas entre colchetes angulares, como "h1" ou "p", e geralmente vêm em pares: uma tag de abertura e uma de fechamento, como "p" e "/p". As tags indicam ao navegador como exibir cada parte do conteúdo, como textos, imagens e links.

Entendendo Comunicação Client x Server.

(Voltar!)
  • O que é o lado cliente?

    • Nada mais é do que a parte em que o dispositivo consome tal tipo de informação na internet através de navegadores, e eles se comunicam através dela com outros computadores, ou seja basicamente computadores interligados usando a internet entre si.

  • O que é o lado servidor(Server)?

    • Como o nome ja sugere, essa é a parte que serve a varios dispositivos praticamente ao mesmo tempo, existe alguns tipos de servidores como: para arquivos, segurança, streaming, e-mail, web.

    Eles podem ser pequenos como um gabinete apenas, ou pode ser de grandes portes como um grande conjuntos de servidores e são chamados de (DatasCenter).

    Todo servidor é baseado em duas partes, a parte lógica que são os (software)como sistemas operacionais, aplicativos, servidores web, ea parte fisica os (hardwares) como discos rigidos, memórias e processadores.

    O que é linguagens de programação?

    (Voltar!)

    Linguagens de programação são ferramentas que permitem escrever instruções para que computadores executem tarefas específicas. Elas usam sintaxes e regras próprias, como se fossem idiomas, para comunicar ao computador o que deve ser feito, desde cálculos simples até a criação de aplicativos complexos.
